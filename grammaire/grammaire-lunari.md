## Introduction

Le **Lunari** est une langue construite de type agglutinant, où les mots sont formés par la combinaison de racines monosyllabiques. Chaque racine porte un sens de base qui peut être modifié ou précisé par l'ajout d'autres racines ou de particules grammaticales.

### Caractéristiques principales
- **Type linguistique** : Agglutinant
- **Ordre des mots** : Sujet-Objet-Verbe (SOV) avec flexibilité
- **Écriture** : Alphabet latin
- **Composition** : Racines monosyllabiques combinées

---

## Phonologie et Écriture

### Voyelles
Le Lunari utilise 5 voyelles de base + 2 diphtongues :
- **Simples** : a, e, i, o, u
- **Diphtongues** : eu, ou

### Consonnes
- **Occlusives** : p, t, k, b, d, g
- **Fricatives** : f, s, z
- **Nasales** : m, n
- **Liquides** : l, r

### Structure syllabique
Les racines suivent généralement ces patterns :
- **CV** : Ma, Li, Ka, Za
- **CVC** : Keu, Peu, Beu, Meu
- **V** : moins fréquent

### Règles de prononciation
1. Chaque lettre se prononce
2. L'accent tonique tombe sur la première syllabe du mot
3. Dans les mots composés, chaque racine garde son accent secondaire
4. **eu** se prononce [ø] comme dans "peu"
5. **ou** se prononce [u] comme dans "doux"

### Règles d'euphonie
Lors de la composition de mots :
- Éviter trois consonnes consécutives (insérer une voyelle de liaison)
- Si deux voyelles identiques se rencontrent, elles fusionnent en une seule
- Préférer l'alternance consonnes/voyelles pour la fluidité

## Morphologie

### Structure des mots

#### Racines simples
Les racines monosyllabiques constituent l'unité de base :
```
Ka = eau
Za = feu
Li = se déplacer
Ma = terre
```

#### Mots composés
Les mots composés se forment par juxtaposition de 2-3 racines :
```
Peu + Za = Peuza (soleil - ciel + feu)
Ka + Li = Kali (nager - eau + se déplacer)
Peu + Ka + Pi = Peukapi (lune - ciel + eau + petit)
```

#### Ordre de composition
L'ordre suit généralement ce schéma :
1. **Élément principal** (premier)
2. **Modificateur/Qualité** (deuxième)
3. **Précision supplémentaire** (troisième, optionnel)

Exemples :
- **Mapeu** = Montagne (Ma-terre + Peu-ciel = "terre qui monte vers le ciel")
- **Zali** = Poisson (Za-feu + Li-mouvement = "mouvement dans l'eau")
- **Kakipeu** = Pluie (Ka-eau + Ki-vent + Peu-ciel = "eau du vent du ciel")

### Catégories grammaticales

#### Pronoms
Les pronoms sont invariables :
- **Mo** = moi, je, me
- **To** = toi, tu, te
- **Lu** = lui, elle, il
- **Gi** = nous, groupe
- **Zi** = eux, elles, ils
- **Keu** = quelqu'un
- **Niu** = personne
- **Pa** = un autre
- **Pi** = d'autres
- **Feu** = chacun

#### Nombres
Les nombres de base :
- **Da** = un (1)
- **Do** = deux (2)
- **Du** = trois (3)
- **Di** = quatre (4)
- **Deu** = cinq (5)

Pour former des nombres plus grands, on utilise des multiplicateurs et des additions :
- **Deu-Da** = six (5+1)
- **Do-Deu** = dix (2×5)
- **Du-Deu** = quinze (3×5)
- **Di-Deu** = vingt (4×5)
- **Deu-Deu** = vingt-cinq (5×5)

## Syntaxe

### Ordre des mots

#### Structure de base : SOV
L'ordre canonique est **Sujet - Objet - Verbe** :
```
Mo Ka Ga-Ba
(Moi eau manger-actif)
= Je bois de l'eau
```

#### Modificateurs
Les adjectifs/descriptifs précèdent le nom :
```
Zabi Zi = grand animal
Kapi Mi = petite plante
```

#### Propositions relatives
Les propositions relatives suivent le nom :
```
Zi [Ma-Peu Bi-Ba] = l'animal [qui est dans la montagne]
```

### Négation
La négation s'exprime par la particule **Boo** :
- Placée après le verbe : négation de l'action
- Doublée : emphase négative forte

```
Mo Ga-Boo = Je ne mange pas
Niu, ZiBoo-Boo = Personne, sans aucune aide
```

### Questions
Les questions se forment par :
1. **Intonation montante** à l'oral
2. **Particule interrogative Keu** en début de phrase pour les questions oui/non
3. **Mots interrogatifs** spécifiques :
   - **KeuBi** = qui ?
   - **KeuDu** = quoi ?
   - **KeuPeu** = où ?
   - **KeuZu** = quand ?
   - **KeuMa** = pourquoi ?
   - **KeuKa** = comment ?

```
Keu To Ga-Ba? = Est-ce que tu manges ?
KeuPeu Lu Bi? = Où est-il/elle ?
```

## Système de Particules

Les particules sont essentielles pour marquer les relations grammaticales :

### Particules verbales

#### Ba (Actif)
Marque le sujet actif de l'action :
```
Mo Li-Ba = Je me déplace (activement)
```

#### Bo (Passif)
Marque le sujet passif ou l'état résultant :
```
Keu Ka-Bo = La lumière s'éteint (devient éteinte)
```

### Particules casuelles

#### Ma (Possession/Génitif)
Marque la possession ou l'appartenance :
```
Mo-Ma Ka = Mon eau
Di-Ma-Gi = Notre sang (sang-possession-groupe)
```

#### Du (Datif/But)
Indique le destinataire ou le but :
```
Mo Zi Du To = Je donne à toi
Li-Ba Du Mapeu = Se déplacer vers la montagne
```

#### Peu (Locatif)
Indique la localisation :
```
Peu Ma = Sur la terre
Peu Beu = Dans l'air
```

#### Ki (Directionnel)
Indique la direction :
```
Ki Peu = Vers le ciel
Ki Ma = Vers le bas
```

#### Za (Causal)
Indique la cause :
```
Za Ka = À cause de l'eau
Za Za = À cause du feu
```

### Particules modificatrices

#### Bi (Étendue)
Indique l'extension, la continuation :
```
Li-Bi = Se déplacer continuellement
Zinu-Bi = Silence étendu
```

#### Beuda (Existence)
Marque l'existence ou la présence :
```
Beuda Zi = Il y a un animal
```

## Formation des Temps

### Système aspectuel
Le Lunari n'a pas de temps grammaticaux comme le français, mais utilise un système aspectuel :

#### Aspect accompli
Marqué par **-Bo** (passif/résultatif) :
```
Mo Ga-Bo = J'ai mangé (état résultant)
```

#### Aspect inaccompli/progressif
Marqué par **-Ba** (actif) :
```
Mo Ga-Ba = Je mange (action en cours)
```

#### Aspect habituel
Marqué par **-Bi** (étendue) :
```
Mo Ga-Bi = Je mange habituellement
```

### Marqueurs temporels lexicaux
Pour préciser le temps, on utilise des adverbes temporels :
- **Ze** = jour, journée
- **Zu** = temps, moment
- **FoZa** = nouveau (Fo + Za)
- **KaNi** = fin, terminaison (Ka + Ni-mort)
- **Ze-Da** = un jour (jour-un)
- **Zu-Zu** = toujours (temps-temps)

```
Ze-Da, Mo Li-Ba = Un jour, je me déplacerai
Zu KaNi = Au moment de la fin
```

## Exemples d'Usage

### Phrases simples

#### Déclaratives
```
1. Mo Zi Bi
   (Moi animal être)
   = Je suis un animal

2. To Keu Ma-Ba
   (Toi lumière voir-actif)
   = Tu vois la lumière

3. Lu Ka Peu Ma Bi
   (Lui/elle eau localisation terre être)
   = Il/elle est dans l'eau sur la terre
```

#### Impératives
L'impératif se forme en omettant le sujet et en utilisant **-Ba** :
```
Ga-Ba! = Mange !
Li-Ba Ki Mapeu! = Va vers la montagne !
```

#### Interrogatives
```
Keu To Du-Ba?
(Question toi savoir-actif?)
= Sais-tu ?

KeuPeu Peuza Bi?
(Où soleil être?)
= Où est le soleil ?
```

### Phrases complexes

#### Coordination
Les propositions sont simplement juxtaposées ou liées par des particules :
```
Mo Ga-Ba, To Gu-Ba
(Je mange, tu dors)
= Je mange et tu dors
```

#### Subordination
La subordination utilise des particules et l'ordre des propositions :
```
Mo Du-Ba [To Li-Ba Peu Mapeu]
(Je sais [tu te-déplaces vers montagne])
= Je sais que tu vas à la montagne
```

### Poésie et style
En poésie, l'ordre des mots est plus flexible :
- Les particules peuvent être omises si le contexte est clair
- Les répétitions sont utilisées pour l'emphase
- Les composés poétiques peuvent être créés librement

```
Exemple du poème existant :
ZinuBi NoMaGi
(Silence-être esprit-possession-groupe)
= Le silence de nos âmes
```

## Règles d'expansion du vocabulaire

### Création de nouveaux mots

#### Par composition
Combiner des racines existantes selon la logique sémantique :
```
Beu (air) + Ki (vent) = Beuki (souffle, brise)
Ma (terre) + Pu (pierre) = Mapu (roche)
```

#### Par dérivation
Ajouter des particules pour créer des variantes :
```
Ga (manger) → Ga-Ba (action de manger) → Ga-Du (nourriture)
Li (se déplacer) → Li-Ma (lieu de déplacement, chemin)
```

#### Emprunts conceptuels
Pour des concepts abstraits, créer des métaphores basées sur des racines concrètes :
```
"Espoir" = KeuLu (lumière-vie)
"Tristesse" = LoSi (ombre-sentiment)
"Mémoire" = DuZu (savoir-temps)
```

### Domaines sémantiques à développer
1. **Émotions et sentiments**
2. **Concepts abstraits** (temps, espace, qualité)
3. **Objets quotidiens**
4. **Actions complexes**
5. **Relations sociales**
6. **Nature et phénomènes**
7. **Corps humain** (détaillé)
8. **Technologie et outils**
9. **Nourriture et cuisine**
10. **Vêtements et parure**

---

## Notes sur l'évolution de la langue

Le Lunari est conçu pour être :
- **Flexible** : nouvelle racines peuvent être ajoutées
- **Logique** : les compositions suivent des patterns sémantiques clairs
- **Expressif** : capable de nuances poétiques et précises
- **Euphonique** : son agréable à l'oreille

Les locuteurs sont encouragés à créer de nouveaux mots composés selon les règles établies, enrichissant ainsi naturellement le vocabulaire de la langue.
